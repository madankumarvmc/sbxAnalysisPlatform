Warehouse Design Platform

Generate comprehensive Excel reports in 4 simple steps

🧭 How to Use This Tool

The application guides you through a sequential 4-step process using the left sidebar navigation. Each step must be completed before moving to the next. The sidebar shows your current step and progress.

📸 Awaiting screenshot: sidebar_navigation.png

⚡ Quick Start:

  1. Follow the sidebar steps sequentially - Start with "Upload Excel File"
  2. Wait for completion - Each step shows validation status
  3. Move to next step - Click the next item only after current step succeeds
  4. Download your report - Final step generates comprehensive Excel analysis

⚠️ Important: Complete each step fully before proceeding. The app validates your data at each stage to ensure accurate analysis.

1

📁 Upload Your Excel File

Drag and drop or browse to select your warehouse data Excel file. The file must contain: OrderData, ReceiptData, and InventoryData sheets.

📸 Awaiting screenshot: step1_upload.png
2

⚙️ Configure Variables

Set your Case-to-Each conversion factors. Default is 12 eaches per case. Adjust based on your product specifications.

📸 Awaiting screenshot: step2_configure.png
3

🚀 Run Analysis

Click the "Run Analysis" button to process your data. Analysis typically completes in 30 seconds for standard datasets.

📸 Awaiting screenshot: step3_run.png
4

📥 Download Excel Report

Download your comprehensive analysis report with 10+ sheets including charts, trends, and actionable insights.

📸 Awaiting screenshot: step4_download.png

⚡ Quick Tips

  • Use the sample file: Test Data/ITC_WMS_RAW_DATA.xlsx for testing
  • Ensure dates are in DD/MM/YYYY format for accurate processing
  • Check that column names match the expected format
  • Large files (>10MB) may take 1-2 minutes to process
  • The output Excel includes both data tables and visual charts
